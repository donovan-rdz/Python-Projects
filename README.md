# Python Projects

A curated collection of Python scripts designed to teach fundamental programming concepts.
Each folder focuses on a specific topic (e.g., conditionals, loops, logical operators) and includes annotated examples and projects.


## Project Structure

```
equation-calculator/

├── src/
│   ├── __init__.py

│   └── ui.py                # Display helpers
├── tests/
│   ├── __init__.py

├── docs/
│   └── usage.md             # Usage examples
├── requirements.txt
└── README.md
```
