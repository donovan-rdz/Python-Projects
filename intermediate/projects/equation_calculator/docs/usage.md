# Usage Guide — Equation Calculator

## Option 1: Evaluate an Expression

You can evaluate any math expression. To use variables, declare them after a comma.

**Examples:**
```
2 + 3 * 4          → 14
2 * x + 1, x=4     → 9
sqrt(16)            → 4.0
2^10                → 1024
```

---

## Option 2: Solve a Linear Equation

Format: `ax + b = c`

**Examples:**
```
2x + 3 = 7         → x = 2.0
-x + 5 = 2         → x = 3.0
3x = 9             → x = 3.0
```

---

## Option 3: Solve a Quadratic Equation

Format: `ax^2 + bx + c = 0`

**Examples:**
```
x^2 - 5x + 6 = 0   → x₁ = 3.0, x₂ = 2.0       (two real roots)
x^2 - 2x + 1 = 0   → x = 1.0 (double root)      (one repeated root)
x^2 + 1 = 0         → x₁ = 0 + 1i, x₂ = 0 - 1i  (complex roots)
```
