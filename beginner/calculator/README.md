# Calculator

A simple terminal-based calculator written in Python.
This project was created as a beginner exercise to practice:
- Python classes
- Conditional statements
- Loops
- Basic input handling

## Features

The calculator supports the following operations:
- Add
- Subtract
- Multiply
- Divide

The program runs in the terminal and allows the user to repeatedly perform operations until choosing the exit option.


### Example
Select operation:
1. Add
2. Subtract
3. Multiply
4. Divide
5. Exit

Choose operation: 1
Enter first number: 5
Enter second number: 3

Result: 8
